## 📜 Project Description
This project is part of the Open Source Software course and involves conducting a structured audit of Git. It includes:

- Understanding Git’s origin, philosophy, and license
- Exploring how Git works in a Linux environment
- Analyzing its ecosystem and community
- Comparing it with proprietary alternatives
- Writing shell scripts to demonstrate Linux and automation skills

## 📂 Repository Contents

| File Name | Description |
|----------|------------|
| script1.sh | Displays system information like kernel, user, uptime |
| script2.sh | Checks if Git is installed and shows details |
| script3.sh | Analyzes disk usage and permissions of directories |
